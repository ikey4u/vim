Title: vim 自定义选项
Date: 2019-01-17
Author: bugnofree
---

安装完成后, 需要在 `$HOME/.vim` 下新建一个 user.vim 文件, 在该文件中配置几个选项,
先看一个样例文件

    let g:pyhome = "/usr/local/opt/python/Frameworks/Python.framework/Versions/3.7"

    let g:terminal = "/Applications/Utilities/Terminal.app/Contents/MacOS/Terminal"
    let g:openex = "open"
    let g:myplan = "$HOME/Documents/sync/myplan.txt"
    let g:mybook = "$HOME/Documents/sync/mybook.txt"
    let g:mytmp = "$HOME/Documents/sync/mytmp.txt"
    let g:myidea = "$HOME/Documents/sync/myidea.txt"
    let g:myidea = "$HOME/Documents/sync/today.txt"

    let g:idapro = "/Applications/IDAPro7.0/ida.app/Contents/MacOS/python"
    let g:idaprotag = "/Applications/IDAPro7.0/ida.app/Contents/MacOS/python/tags"

你只需要修改各个选项的值即可(即等号后面的字符串), 各个选项的含义如下

- g:pyhome

    是 python 的主目录, 可以通过如下命令获得

        python3 -c "import sys; print(sys.executable)"

    比如它输出

        C:\Users\Administrator\AppData\Local\Programs\Python\Python37\python.exe

    那么 python 主目录就是

        C:\Users\Administrator\AppData\Local\Programs\Python\Python37

    然而在 windows 下你需要替换反斜线为斜线

        C:/Users/Administrator/AppData/Local/Programs/Python/Python37

    windows 用户也是一样的.

- g:terminal

    - windows `C:\Windows\system32\cmd.exe`
    - linux 自己找吧
    - mac 参考默认设置

- g:openex

    - windows `start`
    - linux 自己找吧
    - mac `open`

- g:myplan, g:mybook, g:mytmp, g:myidea, g:today

    配置规划, 读书, 临时笔记, 思考, 今日工作的相关文档目录,
    如果你需要则配置, 否则可忽略.

- `idapro`, `idaprotag`

    如果你不知道是啥, 那么你不用管.
